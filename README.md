# 🐾 Визитка кошки — страница и ID-жетон

Минималистичная визитка бенгальской кошки и ID-жетон для шлейки.
Позитивная карточка: фото и контакты для связи, если кошку кто-то встретил.

1. **`index.html`** — минималистичная страница для GitHub Pages, на неё ведёт QR-код с жетона.
2. **Blender-скрипт** для печати компактного ID-жетона на Bambu Lab P1S (AMS): только QR + ушко.

## Как это работает

- На жетоне видны **имя и контакт** — связаться можно сразу.
- **QR** на жетоне ведёт на страницу с фото и контактами.
- Контакты: **Telegram** (`t.me/kirimba256`) и **почта** (`kirimba1024@yandex.com`).

## Структура

```
.
├── index.html            # визитка кошки (GitHub Pages) ← цель QR
├── assets/
│   ├── style.css         # минималистичный тёмный дизайн
│   ├── main.js           # подставляет имя/контакты из data-*
│   ├── favicon.svg
│   └── img/
│       ├── hero.jpg      # главное фото (грузится лениво)
│       └── g01..g11.jpg  # галерея (ленивая подгрузка)
└── blender/
    ├── cat_id_tag.py     # генератор 3D-модели жетона
    └── PRINTING.md       # настройки печати PETG / AMS / QR
```

Фото грузятся **лениво** (`loading="lazy"`), поэтому текст страницы появляется сразу
даже на медленном интернете, а фото подтягиваются по мере прокрутки.

## Что заполнить

1. **`index.html`** — в теге `<main class="page" ...>` поменяй `data-name`, `data-telegram`,
   `data-email`. Фото клади в `assets/img/` (`hero.jpg` — главное, `g01..g11.jpg` — галерея).
2. **`blender/cat_id_tag.py`** — жетон содержит **только QR** (без текста);
   при необходимости поменяй `QR_URL` вверху файла (сейчас `kirimba1024.github.io/cat`).

## Публикация на GitHub Pages

Репозиторий называется **`cat`**, поэтому страница (и цель QR): **`https://kirimba1024.github.io/cat`**.

```bash
git push -u origin main
```

Затем: **Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `(root)` → Save.**
Через минуту страница будет доступна по адресу выше.

> Если переименуешь репозиторий, поменяй `QR_URL` в `blender/cat_id_tag.py` на новый адрес.

## Жетон

См. [`blender/PRINTING.md`](blender/PRINTING.md). Коротко: PETG, сопло 0.4, слой 0.16,
без поддержек, лицом вверх, QR ≥ 28 мм, чёрное на белом.
