# Печать ID-жетона (Bambu Lab P1S + AMS)

## Материал
- **PETG** — предпочтительнее PLA для улицы (UV, влага, тепло в кармане/на солнце).
- Цвета через AMS: **белая база** + **чёрный текст/QR**.

## Настройки слайсера (Bambu Studio)
| Параметр        | Значение           |
|-----------------|--------------------|
| Сопло           | 0.4 мм             |
| Высота слоя     | 0.16 мм            |
| Поддержки       | нет                |
| Ориентация      | лицевой стороной вверх |
| Заполнение      | 15–20% достаточно  |
| Стенки          | 3 периметра        |

## Назначение материалов
| Объект в модели          | Цвет AMS |
|--------------------------|----------|
| `AMS_WHITE_BASE`         | белый    |
| `QR_WHITE_BACKGROUND`    | белый    |
| `AMS_BLACK_TEXT_QR`      | чёрный   |
| `QR_BLACK_RAISED`        | чёрный   |
| `TEXT_*`                 | чёрный   |

## QR-код
- Чёрный на белом — самый надёжный для сканирования. **Не делай белый на чёрном.**
- Не уменьшай QR ниже **28 мм**; надёжнее **30–35 мм**.
- Ссылка короткая (`/cat`), чтобы QR оставался «крупномодульным» и читался камерой телефона.
- После печати — проверь сканирование телефоном при разном освещении.

## Что обязательно на самом жетоне
1. **Имя + контакт** (Telegram / почта) — связаться можно без сканирования.
2. QR на страницу с подробностями (фото, контакты).
