# Печать ID-жетона (Bambu Lab P1S + AMS)

## Материал
- **PETG** — предпочтительнее PLA для улицы (UV, влага, тепло в кармане/на солнце).
- Цвета через AMS: **белая база** + **чёрный текст/QR**.

## Настройки слайсера (Bambu Studio)
| Параметр        | Значение           |
|-----------------|--------------------|
| Сопло           | 0.4 мм             |
| Высота слоя     | 0.12–0.16 мм       |
| Поддержки       | нет                |
| Ориентация      | лицевой стороной вверх |
| Заполнение      | 15–20% достаточно  |
| Стенки          | 3 периметра        |

Чёрный QR теперь **низкий (0.6 мм)** — это весь «верхний/нижний сплошной слой»,
поэтому модули печатаются **на 100% сплошными** (нет рамок с пустым центром)
и **не отрываются**, как было у высоких выпуклых модулей.

## Назначение материалов
Скрипт даёт **два STL**: импортируй плашку, затем `Add part` → загрузи накладку.
| STL-деталь        | Объект    | Цвет AMS |
|-------------------|-----------|----------|
| `qr_1_plate.stl`  | `QR_PLATE` (основание + ушко) | белый  |
| `qr_2_inlay.stl`  | `QR_INLAY` (модули QR)         | чёрный |

## QR-код
- Чёрный на белом — самый надёжный для сканирования. **Не делай белый на чёрном.**
- Модуль **1.5–2.0 мм**, матовый филамент — без бликов, чёткие края модулей.
- Тихая зона (белое поле вокруг QR) уже встроена в плашку — **не обрезай её**.
- Жетон компактный: плашка ровно по QR + маленькое ушко с дыркой сверху.
- После печати — проверь сканирование телефоном при разном освещении.
